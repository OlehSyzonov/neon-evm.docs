## Neon API service

Neon provides several endpoints to assist our users. This page provides the information you need to interact with these services.

## Authentication

No authentication is required. All publically-available JSON RPC endpoints are listed on [Chainlist](https://chainlist.org/?chain=245022926&testnets=true&search=Neon+EVM).

## Neon API specifications

All API calls use the `POST` method.

For example, the following request returns the current Neon Proxy version:

```bash
curl -X POST https://neon-proxy-mainnet.solana.p2p.org \
-H "Content-Type: application/json" \
-d '{"jsonrpc":"2.0","method":"neon_proxyVersion","id":1}'
```

## API services

The following services are available in Neon API V1:

### 1. Version information

- [neon_versions](#get-the-neon-platform-component-versions): returns a list of components provided by the current Neon Platform
- Alternatively, request each component's version individually:
	- [neon_cliVersion](#get-the-neon-cli-version): returns the current version of Neon EVM CLI used by the Neon Proxy
	- [neon_evmVersion](#get-the-neon-evm-version): returns the current version of the Neon EVM program deployed on the Solana
	- [neon_getEvmParams](#get-the-neon-evm-program-version): returns parameters of the Neon EVM program deployed on the Solana
	- [neon_proxyVersion](#get-the-neon-proxy-version): returns the current version of the Neon Proxy
	- [neon_solanaVersion](#get-the-solana-version): returns the version of the Solana node the Neon Proxy runs on


### 2. Gas price information

- [neon_gasPrice](#get-neon-gas-price): returns data on gas price

### 3. Neon transaction data

- [neon_getLogs](#get-transaction-logs): returns transaction log data
- [neon_getTransactionReceipt](#get-transaction-receipt): returns transaction receipt data
- [neon_getSolanaTransactionByNeonTransaction](#get-solana-transactions): returns a list of Solana transactions that correspond to a given Neon transaction
- [neon_emulate](#emulate-a-neon-transaction): returns details of an emultated transaction
